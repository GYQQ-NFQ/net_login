# 广州理工上网 #

## 创建初衷

本脚本创建初衷是为了省下每月的网费(毕竟每个月都扣我30块钱有点心疼)去充月卡

## 使用方法

无脑双击 ***selenium.bat***

## 脚本原理

先通过 *ping* 判断是否有外网连接,如果有则循环 *ping* ,如果没有则进入 *excel_data* 获取账密传递给 *login*


## 进阶玩法

可以通过一些暴力方法获取账密(思路提供 **wifi密码 暴力**)
